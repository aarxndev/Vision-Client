# Vision Client — Remotion Showcase

A ~45 second 1920×1080 promo video for Vision Client, built with [Remotion](https://www.remotion.dev/).

## Scenes

1. **Intro** — Logo, title, v4.0.0 badges
2. **Modules** — Toggle cards and header actions
3. **Netlimiter Bible** — Checkbox presets with apply flow
4. **Live Overlay** — Green slide-in chips (active modules only)
5. **Built-in extras** — Macros, Discord RPC, avatar, WinDivert
6. **Outro** — Logo and discord CTA

## Commands

```bash
cd showcase
npm install
npm run dev          # open Remotion Studio to preview
npm run render       # export out/vision-showcase.mp4
```

Assets (`visionlogo.png`, `visionpfp.png`) live in `public/` and are copied from the parent `assets/` folder.
